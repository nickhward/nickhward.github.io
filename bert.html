CareerBERT

The aim of this project is to employ natural language processing with a BERT model to categorize sentences into specific groups, such as skills, education, and job responsibilities. The model will then assess user resumes for compatibility based on a similarity score

Technology Stack:

  • Python
  • PyTorch
  • Flask
  • Large Language Model
  • Docano (Sentence Labeling)

BERT Model Performance

The model exhibits strong performance. As anticipated, the validation results are slightly lower than the training results, hinting at mild overfitting. However, an 83% validation accuracy remains acceptable. The minor overfitting can be attributed to the size of the dataset.

More importantly, I place a greater emphasis on the F1 score, especially because of the class imbalance. The F1 score is particularly suitable for addressing class imbalances because it provides a balanced measure of precision and recall. In situations with class imbalance, accuracy can be misleading. For instance, if 95% of a dataset belongs to Class A and only 5% to Class B, a model that predicts every instance as Class A would still achieve 95% accuracy, even though it fails to identify any of Class B. The F1 score, on the other hand, considers both false positives and false negatives, offering a more comprehensive view of the model's performance across all classes.
