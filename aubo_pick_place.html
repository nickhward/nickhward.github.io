Aubo Pick And Place Task With Deep Learning

The objective of this project is to develop an efficient method that integrates advanced deep learning techniques, such as pose estimation and computer vision, to enhance pick-and-place operations.

Technology Stack:

  • C++
  • YOLO
  • ROS
  • OpenCV
  • Aruco Tags
  • Grasp Pose Detection

View on GitHub:

Object Detection

Using YOLO v4 object detection algorithm I was able to train a model to recognize and draw bounding boxes on items found in the Lab

Segmentation and Pose Estimation

To accurately determine the pose estimation, we began by segmenting the target object. Once isolated, we executed pose estimation, yielding a precise transformation which the robotic arm could then align to.

Final Results

After completing the aforementioned steps, the robot is now ready to move towards and pick up the object designated by the user.
